## RollingVet - Sistema de Gestión Veterinaria 🐾

## Descripción
Sistema web completo para la gestión de pacientes y turnos de una clínica veterinaria, con panel administrativo y funcionalidades de reserva en línea.

## 🚀 Tecnologías Utilizadas
Frontend
React - Biblioteca principal

React-Bootstrap - Framework de UI

React Router DOM - Navegación

CSS3 - Estilos personalizados
